View time series 1D

 
Multiple time series can be selected and the time history of all of them is plotted, either superimposing all in one plot or plotting them separately but in one figure (ordering them vertically in one plot or in subplots).
 

Select one or more time series
Multiple time series names can be selected from this list. This can be done by standard use of the mouse: a click on the first name and Shift^click on the last name selects all names between the first and the last name; a click on the first name and subsequent Ctrl^clicks on other names select these names including the first name; Ctrl^a selects the whole list.
 

Draw points / lines
The drawing type to be used in the plots, one of the following: points, lines, both.
 

Superimposed in one plot
One plot is drawn having all time series superimposed with different symbols, as shown in the figure below. If up tp 5 time series are selected, they are shown in a legend along with their respective line or/and point type, otherwise a list of the time series names prefixed by the line or/and point type (in matlab syntax) is displayed in the text box of "Running messages". Note that if time series of different range are selected this plot may not give sensible results.

 
 

Vertically translated in one plot
One figure of a single panel is formed with the selected time series displayed with a vertical translation so that they are ordered vertically. The name of each time series is displayed in the left-low corner of the respective area. Note that if many time series are selected the subplots may not be displayed properly. This is the suggested form of plotting segmented or resampled time series all having the same range of values.

 

One subplot per time series
One figure is formed with as many subplots as the selected time series, ordered vertically, where the name of each time series is displayed in the left-low corner of the respective subplot. Note that if many time series are selected the subplots may not be displayed properly. This is the suggested form of plotting if time series of different range are to be selected.

 
 

Running Messages
Messages about the time series being plotted are displayed here.
 

Exit
The window is closed.
