Segment Time Series
Multiple time series can be selected and each selected time series is split to segments of equal length, allowing the specification of the sliding step.
 

Current List of Time Series
Multiple time series names can be selected from this list. This can be done by standard use of the mouse: a click on the first name and Shift^click on the last name selects all names between the first and the last name; a click on the first name and subsequent Ctrl^clicks on other names select these names including the first name; Ctrl^a selects the whole list.
 

Length of segment (n)
The length of each segment the time series is split to. If this is larger than the time series length no operation will be done on this time series.
 

Sliding step (s)
The step of the sliding window. Default is the value of the length of segment (n), which means that the segments are consecutive. A values s<n indicates overlapping whereas s>n leaves gaps of data not included in the segments.
 

Ignore remaining data
Two modes can be selected: Beginning (B) or End (E). This option is in action when the splitting of the time series with sliding windows leaves an unused data window. Then this window of unused data is from the beginning or end of the time series according to the chosen mode.
 

OK
By pressing this button, the segmenting of the selected time series starts and when it is completed the window is closed.
 

Cancel
The window is closed without segmenting any time series.
