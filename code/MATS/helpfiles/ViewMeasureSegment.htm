Plot of measures vs segments

 
A selection of measures computed on segments of time series, i.e. time series with names containing indices after the character 'S', can be plotted against the segment indices in a figure window. This type of figure allows to track changes in the evolution of measure(s) along the recording epoch split in the time resolution of the duration of each segment.

Two or more segments can be selected from the time series list having the title Select segments of a time series. The segments of a time series are identified by the time series name, the specified parameters of the length of segment (n), sliding steps (s), the character 'B' or 'E' for ignoring remaining data, and the indices after the character 'S' (if there is a string after the index this should be the same in the names of the segments). If no such segments names exist the list of segments is empty and this operation cannot run. Further, one or more measures can be selected from the measure list having the title Select one or more measures. For measures defined in terms of one or more measure specific parameters, the measure name regards a specified set of parameters (one parameter value for each name). A subset of segment and measure names can be selected from the respective lists with standard use of the mouse: a click on a name and Shift^click on a another name further down in the list selects all names between the former and the latter; a click on a name and subsequent Ctrl^clicks on other names select these names including the first name; Ctrl^a selects the whole list.

Tips: For the selection of a subset of segment time series names from the list, click on the segment time series of this subset appearing first in the list and then Shift^click at the last time series in the list. In this way, all time series from the first selected to the last time series in the list will be marked and from those only the ones that have the same segment time series name as the first and differ only to the segment index will be processed.

 

View selected plot
Provided that a list of at least two segment names and one or more measures names are selected, by pressing this button a separate figure window will be displayed (as the one shown below). The graph of each selected measure vs the segment indices is displayed with a distinct point or line type. When up to 5 measure names are selected, they are listed in a legend together with their corresponding drawing type, otherwise they are listed in the "Running messages" text box.
 

draw lines / points
The user can choose one of the following: Lines, Points, Both. This is the drawing type for the measure vs segment index plot and a different drawing type is used for each measure, as explained in the "Running messages" text box.
 

Running Messages
A text box displaying messages for the running operations.
 

Exit
By pressing this button the window of "Plot of Measures vs Segments" will disappear and the user will be moved to the "View Measures" window. Note that any created figures will remain until the user exits from the "View Measures" window.
