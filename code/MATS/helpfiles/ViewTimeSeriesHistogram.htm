View time series histogram

 
Multiple time series can be selected and the histogram for each of them is plotted, either superimposing all in one plot or plotting them separately in figures containing matrix plots of a given size (subplots organized in rows and columns in each figure).
 

Select one or more time series
Multiple time series names can be selected from this list. This can be done by standard use of the mouse: a click on the first name and Shift^click on the last name selects all names between the first and the last name; a click on the first name and subsequent Ctrl^clicks on other names select these names including the first name; Ctrl^a selects the whole list.
 

Number of bins
For the histograms the equidistant binning method is used and the number of bins is given here. If the default value 0 is selected, then the number of bins is set to sqrt(n/5), where n is the length of each selected time series.
 

Superimposed in one plot
One plot is drawn having the lines of the histograms of all selected time series superimposed with different line type, as shown in the figure below. If up to 5 time series are selected, they are shown in a legend along with their respective line type, otherwise a list of the time series names prefixed by the line type (in matlab syntax) is displayed in the text box of "Running messages". Note that the y-axis shows the frequencies, so if the selected time series vary significantly in their length, the lines are not at the same level and may not be comparable.

 

 

One subplot per histogram
When using this option, one histogram will be displayed at each subplot for each of the selected time series. The subplots are organized in rows and columns, which are determined by the user (see below). Depending on the number of selected time series and the given rows and columns, more than one figure may be generated. The histogram at each subplot can be displayed as bars or line and the normal curve can be selected to be superimposed on it. See the figure below for an example on 6 histograms in 2 x 3 matrix plot. If the user chooses to display a normal curve also the p-value of the Kolmogorov-Smirnov test for normality is shown in the label of the horizontal axis.

 

 
 
Subplots per row
The number of rows for the subplots in each figure.
 
Subplots per column
The number of columns for the subplots in each figure.
 
Display normal curve
If selected a red line of the bell-shaped normal curve (with the same mean and standard deviation as the selected time series) will be superimposed on the histogram.
 
Draw type
The drawing type to be used in the histogram plots: bars or line.
 

Running Messages
Messages about the histograms being plotted are displayed here.
 

Exit
The window is closed.
