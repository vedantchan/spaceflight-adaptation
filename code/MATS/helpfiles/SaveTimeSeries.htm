Save Time Series

 
Multiple time series can be saved to files with the respective time series names at the selected directory. This operation can be repeated, so that different directories can be chosen.

The format of the data files can be plain text (ascii), excel (xls) and matlab data files (mat). Note that in each data file a single time series is saved with the name of the time series and a suffix determined by the selected format.

 

Select one or more time series ...

Multiple time series names can be selected from this list. This can be done by standard use of the mouse: a click on the first name and Shift^click on the last name selects all names between the first and the last name; a click on the first name and subsequent Ctrl^clicks on other names select these names including the first name; Ctrl^a selects the whole list.
 

Select directory
This button opens a matlab dialogue window to specify the directory path.
 

Data file format
One of three modes of data format can be selected: plain text (ascii), excel (xls) and matlab data files (mat). The selected format is used for all the selected time series.
 

Save selected scalar time series
By pressing this button the selected time series will be attempted to be saved to data files with the same names and a suffix of the specified format.
 

Running messages
Messages about the files being saved are displayed here.
 

Exit
The window is closed.
